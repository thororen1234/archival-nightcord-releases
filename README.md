# Archived Nighcord Releases 
> ⚠️ **This repository exists solely to document known malicious releases.**
> Do not install or run any code from this archive.

## Overview

Versions **v1.18.2** and **v1.18.3** of nightcord were found to contain token logging functionality that exfiltrated user authentication tokens to a third-party server.

This archive is preserved for:
- Verification that the malicious versions existed
- Community reference and transparency
- Allowing affected users to confirm exposure

## If You Used v1.18.2 or v1.18.3

1. Assume your token was compromised
2. Immediately invalidate/regenerate your token
3. Check for unauthorized activity on your account
